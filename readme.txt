To generate pdf version of the website https://www.comp.nus.edu.sg/~cs1101s/sicp/chapters/<number>
run scrip_scrap.py 

ScipFiles folder stores individual chapters and can be safely deleted if you want.
All generated pdfs can be deleted safely
Please do not delete any other files though

Needs selenium, bs4, pdfkit, PyPDF2 installed to work properly.
